# Scoop 存储桶: `bughug`

本代码库 (`bughug/scoop`) 提供一系列 Scoop 应用清单。

## 使用此存储桶

1.  **添加存储桶:**
    ```powershell
    scoop bucket add bughug https://github.com/bughug/scoop
    ```

2.  **安装应用程序:**
    ```powershell
    scoop install bughug/<应用名称>
    ```
    (请将 `<应用名称>` 替换为 `bucket` 目录中的应用清单名称。)
